WhatsApp Gateway API

Dokumentasi resmi untuk integrasi WhatsApp Gateway MikBill dengan aplikasi pihak ketiga.

Endpoint: Send Message

Gunakan endpoint ini untuk mengirim pesan WhatsApp ke nomor tertentu.

POST /api/send-message

Request Parameters (JSON)
Parameter Type Required Description
api_key String Yes Kunci API rahasia Anda. Ambil dari Admin Panel.
number String Yes Nomor tujuan (contoh: 081234567890 atau 6281234567890).
message String Yes Isi pesan yang akan dikirim.
Example Request (cURL)
curl -X POST http://your-domain.com/api/send-message \ -H "Content-Type: application/json" \ -d '{ "api_key": "YOUR_SECRET_API_KEY", "number": "08123456789", "message": "Halo, ini pesan percobaan dari API." }'
Example Response (Success)
{ "status": "success", "message": "Message processed", "data": { "status": true, "response": "Sent via Gateway" } }
Example Response (Error)
{ "status": "error", "message": "Unauthorized: Invalid API Key" }
Notes
  • Pastikan WhatsApp Gateway dalam keadaan CONNECTED di Admin Panel.
  • API Key bersifat rahasia. Jangan bagikan kepada pihak yang tidak berwenang.
  • Jika API Key bocor, segera Generate Key Baru di menu WhatsApp Admin Panel.
